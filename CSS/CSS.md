# what is css ?

CSS is a style sheet language used for describing the look and formatting of a document written in a markup language.

# To write CSS we need to follow structure

selector {
    property: value;
}

# what is css selector ?

CSS selector is used to select the element or elements you want to style.

# CSS in HTML

There are 3 types of CSS in HTML

1. Inline CSS
2. Internal CSS
3. External CSS

# Inline CSS

Inline CSS is used to apply styles to a single element.

# Internal CSS

Internal CSS is used to apply styles to a single HTML document.

# External CSS

External CSS is used to apply styles to multiple HTML documents.
