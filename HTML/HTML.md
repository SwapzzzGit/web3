# What is pre tag ?

Pre tag is used to define preformatted text.  //JAISA LIKHOGE VAHI DIKHAYGA

# What is code tag ?

Code tag is used to define a piece of computer code. //CODE KA KISI BHI FORMAT NHI CHALTA

# What is audio tag ?

Audio tag is used to define sound content.

# What is video tag ?

Video tag is used to define video content.

# What is target tag ?

Target tag is used to define where to open the linked document.

# What is controls tag ?

Controls tag is used to define extra controls for the audio and video.

# What is autoplay tag ?

Autoplay tag is used to define that the audio or video will start playing as soon as it is ready.

# What is form tag ?

Form tag is used to create a form for user input.

# What is input tag ?

Input tag is used to create an input field for user input.

# What is label tag ?

Label tag is used to define a label for an input element.

# What is checkbox tag ?

Checkbox tag is used to define a checkbox for user input.

# What is radio tag ?

Radio tag is used to define a radio button for user input.

# What is date tag ?

Date tag is used to define a date picker for user input.

# What is file tag ?

File tag is used to define a file upload for user input.
